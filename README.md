# use-pwa2

use-pwa2 is install and update handler for pwa.

## Features

- TypeScript support
- SSR support
- Update support

## Installation

`npm i --save use-pwa2`

## Example

[Example](https://use-pwa2.kk-web.link/)

## Usage

[Example Code](https://github.com/piro0919/use-pwa2/blob/master/src/App.tsx)

## Return

| Return         |   Type   |
| -------------- | :------: |
| enabledInstall | Boolean  |
| enabledUpdate  | Boolean  |
| installPwa     | Function |
| isLoading      | Function |
| updatePwa      | Function |
